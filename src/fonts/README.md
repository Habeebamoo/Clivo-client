# Fonts

Place the following `.ttf` font files in this directory.
They are referenced in `app/globals.css` via `@font-face`.

- Inter_18pt-Medium.ttf
- OpenSans-Regular.ttf
- DMSans-Medium.ttf
- Exo-Medium.ttf
- Outfit-Regular.ttf
- Outfit-Medium.ttf
- PlusJakartaSans-Light.ttf
- PlusJakartaSans-Medium.ttf
- Roboto_Condensed-Regular.ttf
